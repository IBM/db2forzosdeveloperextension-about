# {{ page.title }}

You can change the user IDs, passwords, or both for multiple connections profiles at once.

To update the user IDs, passwords, or both:

1. From the **DB2 FOR Z/OS CONNECTIONS** view, click the **Manage Connections Credentials** icon.

   ![Manage connections credentials icon]({{site.baseurl}}/assets/images/manage-connection-creds-icon.png)

2. Select a connection profile, or multiple connection profiles, that you want to change the user ID, password, or both for.

   ![Change User and Password in Connection Profile dialog]({{site.baseurl}}/assets/images/change-user-pass-connection-profiles.png)

3. In the **Connection Details** section, enter a new user ID, password, or both.

4. Click **Apply**.

The result is that the updated user IDs and passwords are used when you reconnect to those databases.
